Conjuntos de dados e datasets

Os conjuntos de dados e os datasets formam a base para realização de qualquer análise de dados no mundo do trabalho, bem como no mundo acadêmico. Conjunto de dados pode ser representado por diversas planilhas, bancos de dados e etc.. Já o dataset é mais específico, todos os dados já estão presentes em tabela única. A organização dos dados para realização do tratramento estatístico é estruturado de modo que cada coluna representa uma variável particular e cada linha corresponde a um determinado membro do conjunto de dados. Cada valor da combinação coluna e linha é um dado.

R é um ambiente de software livre para computação e gráficos estatísticos. Compila e roda em uma ampla variedade de plataformas UNIX, Windows e MacOS.
